ReadMe
- Launch the terminal.
- Change directory to the project folder using the "cd" command.
- Access the source code folder "src" using the "cd" command.
- Compile the code by running the command "javac Server.java".
- Execute the program by typing "java Server" followed by a port number and a directory path.
For instance, typing "java Server 8080 public" will start the program on port 8080
and use the "public" folder as the server directory.
Note that the directory should be located outside the "src" folder.
- Open a web browser and type "localhost:port number/" to access the main index site.
